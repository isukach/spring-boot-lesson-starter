1. Создать Spring Boot приложение для хранения тектовых сообщений с использованием Spring Data репозиториев и Rest API,
подключить devtools и actuator.
2. Приложение должно уметь принимать запросы вида /messages/1, получать сообщения из базы и отдавать их пользователю
в формате JSON.
3. Добавить REST-endpoint для сохранения нового сообщения в базу.
